Pytanie: czy mo\.zliwe jest obliczanie bez bitów — bez zer i jedynek,

Funkcja rezonansowa i trajektoria

Podstawowy element modelu to funkcja rezonansowa:

R_n(x, y) = 2 x^n yx^2n + y^2

Dla dowolnego <em class='math'>n > 0 i <em class='math'>x, y > 0 wartość <em class='math'>R_n in (0, 1).

Trajektoria rezonansowa to krzywa <em class='math'>y = x^n, wzdłu\.z której <em class='math'>R_n = tfrac12

dokładnie dla ka\.zdego <em class='math'>x — funkcja osiąga szczyt w sposób przestrzennie

ciągły, bez dyskretnych progów.

Rodzina rezonatorów pozwala na przestrajanie szczytu:

R_a,b(x) = x^2nx^an + x^bn,

szczyt = 1k b = 2 + łog(k-1)n łog x

Wartości graniczne <em class='math'>0 i <em class='math'>1 nie są osiągalne — pole nigdy nie jest

w pełni ,,zerowe'' ani ,,nasycone''.

Uzupełnienie amplitudowe: <em class='math'>R + barR = 1,

co domyka przestrzeń logiczną bez wartości binarnych.

Wspólny mianownik wszystkich wariantów: trajektoria <em class='math'>n wyznacza kształt pola.

Zmiana <em class='math'>n to zmiana logiki — nie zmiana jednej komórki pamięci,

lecz całej geometrii obliczeń.
